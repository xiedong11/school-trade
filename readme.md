### 校园淘宝说明文档
校园淘宝项目给在在校大学生提供一个很好的校园商品交易平台，
内置校园跳蚤市场，动态社区，失物招领，等模块
可支持扩展成多种场景校园社交类app定制，方便小伙伴们二次开发


#### 项目技术点：

- 1.内置登录注册功能，用户可注册登录使用

- 2.BmobFile开发跳蚤市场图片存储功能，提供用户浏览商品，发布商品功能

- 3.购买记录查询，购物车，收藏商品，模拟商品交易

- 4.方便开发者进行二次开发定制
